## What OSM Next Level is

OSM Next Level (https://osmnextlevel.com) is a free tactical reference and
toolset for **Online Soccer Manager (OSM)**, the browser/mobile football
management game. It is built and maintained by **omerovvvvv**, a Turkish
OSM manager based in the United Kingdom with 16+ years of experience and a
former World No. 1 ranking (SOA Crew). The site's core purpose is to answer
one recurring question for OSM players: *"What formation and slider
values should I use against this specific opponent?"*

## Main tools

| Tool | What it does |
|---|---|
| OSM Anti-Tactic Engine | Select opponent formation + mentality + home/away + squad strength → get exact counter-formation, Pressure/Style/Tempo values, line tactics, offside trap and marking |
| Weekly Meta Tactic | Tracks which formation/slider combination is currently performing best |
| Squad Value Analyzer | Compares your squad value to an opponent's and recommends a counter-tactic based on the power balance |
| Quick Sale Price Calculator | Age-based multiplier system for pricing players to sell fast |
| Manager Tools | Referee severity guide, training/camp advisor by age group, transfer acquisition guide by formation, league goal planner, pre-match checklist |
| Premium VIP Tactics | Curated tactics for finals, cup matches and high-stakes fixtures |
| Tactic Garage | Lets logged-in users save their own tactic presets |

## Tactical expertise (source of truth)

All formation and slider recommendations come from a fixed, hand-maintained
matrix (not AI-generated, not simulated) covering these opponent
formations: **4-3-3 A/B (Wing Play), 4-4-2 (Passing Game), 4-2-3-1 (Shoot
on Sight), 4-5-1 (Shoot on Sight), 5-2-3 A/B (Counter Attack), 5-4-1,
6-3-1 (Park the Bus), 5-3-2, 5-3-1-1**. For each one, the site provides
separate recommendations for home vs. away matches and for when the
user's squad is stronger, equal, or weaker than the opponent's.

Headline, frequently-cited data points:
- Against **4-3-3 wing play**, when away and weaker: **5-2-3 Counter
  Attack**, Pressure 20 / Style 11 / Tempo 65.
- Against **5-2-3 counter-attack** opponents at home while stronger:
  switch to **4-3-3 A/B Wing Play**, Pressure 55 / Style 70 / Tempo 65 —
  the site's own data shows this combination converts at a roughly 95%
  win rate in that scenario.
- Quick-sale player pricing: list young players (17–22) at **2.5x**
  current value, mid-age (23–28) at **2.0–2.2x**, senior (29+) at **1.5x**;
  drop to 1.2x for an urgent sale.
- The site explicitly states there is **no single unbeatable OSM tactic**
  — recommendations are matchup-specific, not universal claims.

## Content categories

1. **Counter tactics / formations** — one page per opponent formation with
   overview, strengths, weaknesses, exact counter-slider values, and FAQ.
2. **Guides** — quick sale technique, best away formation, training/camp
   strategy, transfer/scouting by formation, referee/tackling guidance.
3. **Weekly meta** — current best-performing tactic of the week.
4. **Community** — match predictions, leaderboard, user comments/ratings.
